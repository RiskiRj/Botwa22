# Ganz Bot
Ganz bot adalah bot whatsapp hasil pe-nggarapan ulang dari RiskiRj/stikerinbot yang sudah di edit agar bisa di jalankan di termux.

# Termux

```
apt install ffmpeg
apt install imagemagick
apt install nodejs
apt install git

cd Ganzbot
npm i
